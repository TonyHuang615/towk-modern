# TOWK-Modern 项目概述

## 项目背景

**客户**: 新加坡东安会馆 (Tung On Wui Kun)  
**成立时间**: 1876年  
**现有网站**: https://towk.sg

新加坡历史最悠久的华人宗乡社团之一，为祖籍中国广东东莞宝安两县移民的地缘组织。

## 项目目标

1. **视觉升级** — 现代化设计，提升品牌形象
2. **功能增强** — 会员系统、活动管理、在线报名
3. **管理便捷** — 后台CMS，非技术人员维护
4. **移动优先** — 完美适配所有设备
5. **多语言支持** — 中文、英文双语

## 核心改进点

### 1. 视觉体验
- 现代化设计语言
- 响应式布局
- 流畅动画效果
- 高清图片展示

### 2. 内容管理
- 可视化CMS后台
- 文章/活动/相册管理
- 权限分级
- 富文本编辑器

### 3. 会员功能
- 会员注册/登录
- 会员资料管理
- 会员专属内容
- 活动报名

### 4. 活动系统
- 活动发布与管理
- 在线报名
- 活动提醒
- 历史归档
